# T-PACHIN 柏青哥機台資訊平台

![T-PACHIN Logo](https://static.wixstatic.com/media/d5e540_e347f7f914a14623b91140a29b40cb08~mv2.png)

這是一個柏青哥/柏青嫂機台資訊展示平台，提供機台查詢、收藏、保留、模擬遊玩等功能。

## ✨ 功能特點

| 功能 | 說明 |
|------|------|
| 🎰 **機台列表** | 瀏覽各類柏青哥機台，支援分類篩選、廠商篩選 |
| ❤️ **收藏功能** | 將喜愛的機台加入收藏，跨頁面同步 |
| ⏰ **保留倒數** | 機台保留時間倒數計時，即時更新，到期提醒 |
| 🎮 **模擬器** | 柏青斯洛機台模擬器，支援上分/下分、保留功能 |
| 📋 **公告系統** | 系統公告、站內信、玩家Q&A |
| 👤 **個人中心** | 會員資訊、收藏/保留管理、注單明細 |

## 📱 頁面說明

| 頁面 | 連結 | 說明 |
|------|------|------|
| **首頁** | [index.html](index.html) | 機台總覽與分類瀏覽，支援篩選 |
| **機台列表** | [machines.html](machines.html) | 我的收藏、保留中、玩過記錄 |
| **公告中心** | [news.html](news.html) | 系統公告、信件、Q&A |
| **個人中心** | [about.html](about.html) | 會員資料、收藏/保留管理、注單明細 |
| **模擬器** | [simulator.html](simulator.html) | 🎰 柏青斯洛模擬器，體驗機台遊玩 |

## 🛠️ 技術棧

- **HTML5** - 頁面結構
- **Tailwind CSS** - 樣式設計（CDN引入）
- **Font Awesome 4.7** - 圖示系統
- **JavaScript (原生)** - 互動功能
- **LocalStorage** - 跨頁面狀態同步

## 🔄 跨頁面同步功能

本網站使用 `localStorage` 實現以下狀態的跨頁面同步：

- ❤️ **收藏機台**：在任一頁面收藏/取消，所有頁面即時更新
- ⏳ **保留機台**：保留狀態與倒數時間全域一致
- 💰 **點數系統**：會員點數跨頁面同步
- 👤 **會員資訊**：暱稱、頭像跨頁面同步

## 🎮 模擬器功能

- 點擊機台卡片「進入遊戲」可前往模擬器
- 支援上分/下分（1:1 兌換，最低 100 代幣）
- 保留功能：根據遊玩轉數計算保留時間
- 畫中畫（PiP）支援
- 機台數據圖表即時更新

## 🚀 部署說明

此為靜態網站，可部署於任何靜態網頁託管服務：

### GitHub Pages 部署步驟

1. 將所有檔案上傳至 GitHub 倉庫
2. 進入倉庫 Settings → Pages
3. 選擇 main 分支作為來源
4. 等待幾分鐘即可訪問

## 💡 使用提示

- 點擊機台卡片上的 ❤️ 可加入收藏
- 保留中的機台會顯示倒數計時
- 點擊個人頁的 VIP 徽章可查看特權說明
- 點擊大頭照可切換頭像
- 點擊機台「進入遊戲」即可前往模擬器體驗
